# Motion

Motion helps people understand state.

Motion is not decoration.

## Duration Scale

Use short durations:

- `duration.fast`: about 100-150ms.
- `duration.normal`: about 180-240ms.
- `duration.slow`: about 300-400ms.

Longer motion needs a reason.

## Easing

Use calm easing.

Avoid bouncy or playful easing in Studio.

## Allowed Uses

- Page transition.
- Modal open/close.
- Toast appearance.
- Preview refresh.
- Save state.
- Error appearance.
- Focus movement.

## Reduced Motion

Respect `prefers-reduced-motion`.

Reduced motion should remove or shorten non-essential movement.

## Studio Rule

Studio motion must make state clearer:

- Saving.
- Saved.
- Failed.
- Preview refreshed.

It must not slow repeated work.
