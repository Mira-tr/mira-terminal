# Modal

## Purpose

Modal interrupts for focused action or confirmation.

Use only when the user must decide before continuing.

## Variants

- Confirm.
- Preview.
- Import preview.
- Danger confirmation.
- Help.

## States

- Opening.
- Active.
- Loading.
- Error.
- Closing.

## Keyboard

- Focus moves inside on open.
- Focus is trapped.
- Escape closes when safe.
- Focus returns to opener.

## Accessibility

- Modal has title.
- Modal has description when needed.
- Background is inert.

## Mobile Behavior

- Fits viewport.
- Content scrolls inside.
- Primary actions remain reachable.

## Brand Public

Use rarely.

## Creator Public

Use rarely.

## Studio

Use for import preview, danger confirmation, and focused editing.
