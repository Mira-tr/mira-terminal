# Button

## Purpose

Button performs an action.

Use links for navigation.

## Variants

- Primary.
- Secondary.
- Ghost.
- Danger.
- Icon button.

## States

- Default.
- Hover.
- Active.
- Focus-visible.
- Disabled.
- Loading.
- Success.
- Warning.
- Error.

## Keyboard

- `button` element.
- Enter and Space activate.
- Disabled button must not receive action.

## Accessibility

- Visible focus.
- Text label or accessible name.
- Loading state announced when important.

## Mobile Behavior

- Minimum 44px touch target.
- Wrap text rather than overflow.

## Brand Public

Use one primary CTA per section.

## Creator Public

Use creator theme tokens.

## Studio

Use compact buttons.

Danger buttons must be visually separated from normal actions.
