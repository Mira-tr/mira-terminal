# Status Indicator

## Purpose

Status Indicator shows current state at a glance.

## Variants

- Success.
- Warning.
- Error.
- Info.
- Draft.
- Private.
- Public.
- Archived.
- Loading.

## States

- Default.
- Updating.
- Stale.
- Error.

## Accessibility

Use text plus icon or shape.

Do not use color alone.

## Mobile Behavior

Status text can wrap.

Avoid tiny dots without labels.

## Brand Public

Use lightly for public status.

## Creator Public

Use for creator works or collection status.

## Studio

Use heavily for validation, backup, build, publish, and Git read-only state.
