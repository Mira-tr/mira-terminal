# Card

## Purpose

Card groups one item or small idea.

Do not use cards for every page section.

## Variants

- Feature card.
- Mini card.
- Status card.
- Creator card.
- Studio panel.

## States

- Default.
- Hover when clickable.
- Focus-visible when interactive.
- Selected.
- Disabled.
- Empty.

## Keyboard

If card is clickable, it must contain a real link or button.

Do not make a non-interactive card look clickable.

## Accessibility

- Use headings inside cards.
- Avoid entire-card click if it hides controls.

## Mobile Behavior

- Stack cards.
- Preserve readable padding.

## Brand Public

Use cards sparingly.

## Creator Public

Cards can show works or activity.

## Studio

Use panels for status and forms, not decorative cards.
