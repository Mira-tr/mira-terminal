# Link

## Purpose

Link navigates to another place.

Do not use a link for a destructive action.

## Variants

- Text link.
- Navigation link.
- External link.
- Back link.
- Related link.

## States

- Default.
- Hover.
- Active.
- Focus-visible.
- Visited.
- Disabled when represented as non-link text.

## Keyboard

- `a` element with valid `href`.
- Enter activates.

## Accessibility

- Link text should explain destination.
- External links should indicate external behavior when needed.

## Mobile Behavior

- Adequate tap area.
- Avoid tiny inline links near destructive actions.

## Brand Public

Use editorial links with clear destination.

## Creator Public

Use links for creator navigation and back to RELMUA.

## Studio

Use links for navigation only.

Do not mix link and tab meaning.
