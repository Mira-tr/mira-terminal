# Responsive Specification

Responsive design must be verified at:

- 375
- 390
- 430
- 768
- 1024
- 1440

## Checkpoints

At every width, check:

- Header.
- Navigation.
- CTA.
- Table.
- Modal.
- Sidebar.
- Breadcrumb.
- Path.
- Long text.
- Tags.
- Empty state.
- Danger Zone.

## 375 / 390 / 430

Rules:

- Single column for most layouts.
- No horizontal scroll.
- CTA can wrap.
- Touch targets remain large.
- Modal fits viewport.
- Tables transform or scroll intentionally.

## 768

Rules:

- Two columns only when content remains readable.
- Sidebar may collapse.
- Breadcrumb should wrap cleanly.

## 1024

Rules:

- Studio can show sidebar and workspace.
- Public content still uses max width.
- Tables can show more columns.

## 1440

Rules:

- Do not stretch prose.
- Use max width.
- Studio can use multi-panel layout.
- Keep primary work area readable.
