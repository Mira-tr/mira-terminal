# Creator Public Surface

Creator Public expresses an individual creator.

It shares system rules but can use different theme tokens.

## Common Structure

Creator pages should share:

- Header pattern.
- Creator navigation.
- Back to RELMUA.
- Content sections.
- Accessibility rules.

They should not copy HTML and components per creator.

## Theme Token Difference

Creator identity should come from:

- Color tokens.
- Typography accent.
- Surface texture.
- Spacing mood.
- Visual assets.

Not from duplicating component logic.

## Chikage Direction

Chikage should feel:

- Modern Japanese.
- Paper.
- Ink.
- Indigo.
- Seasonal.
- Quiet.

Avoid:

- Shrine template.
- Ryokan template.
- Period drama.
- Japanese fantasy UI.

## Creator Rule

Creator Public must not look like Brand Public with only the name changed.

It must also not become only one module such as TRPG.
