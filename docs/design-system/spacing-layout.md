# Spacing and Layout

Spacing creates calm and hierarchy.

It also prevents accidental clicks.

## Spacing Scale

Use a 4px base scale.

Recommended values:

```text
4 / 8 / 12 / 16 / 24 / 32 / 48 / 64 / 96 / 128
```

## Public Layout

Brand Public:

- Max page width: comfortable editorial width.
- Hero width: wide but not stretched.
- Reading width: narrow enough for Japanese prose.
- Section spacing: generous.

Creator Public:

- Slightly more intimate than Brand.
- Uses theme tokens for personality.
- Keeps common structure.

## Studio Layout

Studio / Admin:

- Uses denser spacing.
- Avoids huge vertical gaps.
- Keeps forms and lists efficient for PC input.
- Uses panels and tables when repeated work matters.

Admin content width should fit common desktop work without forcing constant
scrolling.

## Grid Gaps

Brand cards:

- Medium gap.

Creator sections:

- Medium to relaxed gap.

Studio lists:

- Compact gap.

## Breakpoints

| Width | Rule |
| --- | --- |
| 375 | Single column. Avoid horizontal scroll. |
| 390 | Primary mobile verification width. |
| 430 | Large phone, still single column for complex UI. |
| 768 | Tablet. Two columns only when content remains readable. |
| 1024 | Small desktop/tablet landscape. Studio may show sidebar. |
| 1440 | Desktop. Cap content width; do not stretch text endlessly. |

## Mobile Spacing

Mobile should reduce spacing, not remove it.

Buttons must remain tappable.

Danger Zone must remain visually separated.
