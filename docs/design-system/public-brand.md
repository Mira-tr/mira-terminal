# Brand Public Surface

Brand Public is the RELMUA brand site.

It should be calm, modern, and easy to understand in three seconds.

## Character

- White based.
- Quiet.
- Modern.
- Readable.
- Spacious.
- Information first.

## Visual Hierarchy

Brand Public should show:

1. What RELMUA is.
2. What can be seen next.
3. One clear next action.

## Components

Use fewer components.

Prefer:

- Editorial sections.
- Feature blocks.
- Clear links.
- Calm cards.

Avoid:

- Dashboard density.
- Too many equal cards.
- Internal technical language.

## Empty State

Empty states should explain curation.

Example:

```text
Only public-ready tools are shown here.
```

## Navigation

Navigation should show current page and next meaningful path.

Do not expose Creator module internals as global categories.
