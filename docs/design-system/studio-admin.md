# Studio / Admin Surface

Studio / Admin is a production tool.

It is not a public marketing page.

## Character

- PC input first.
- Dense enough for daily work.
- Status visible.
- Mistake resistant.
- Clear language.
- Comfortable for long use.

## Layout

Studio should use:

- Sidebar.
- Workspace.
- Inspector.
- Preview.
- Activity.
- Status.
- Toolbar.

Public-style huge white space should not be copied into Studio.

## Density

Studio must fit repeated work:

- Tables.
- Compact forms.
- Validation summary.
- Inline status.
- Clear save state.

## Safety

Dangerous operations must be visually separated.

Users must see:

- Impact.
- Count.
- Backup recommendation.
- Confirm.
- Cancel.
- Rollback when possible.

## Language

Beginner mode uses human explanations.

Advanced mode may show paths, JSON, manifest, and diagnostics.
