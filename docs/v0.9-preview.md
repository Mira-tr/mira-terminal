# v0.9 Preview運用メモ

## 状態

Home / About / TRPG Scenario / House Rules / Game Works / Tools / Notesと、それぞれのAdmin管理・Public Export・Backup運用が利用できます。

## 公開範囲

- GitHub Pagesへ公開するのはdist/のみ
- dist/はapps/web/から生成
- apps/admin/は非公開
- Backup JSONは非公開
- Public JSONだけをapps/web/.../data/へ配置

## データ運用

AdminはlocalStorage管理です。ブラウザデータ消去や端末変更に備え、各モジュールのBackup Exportを定期実行します。

## TRPG年齢区分

全年齢 / R18の2択です。R18G等の旧値はR18へ統合し、細かな注意はタグで扱います。

## Preview公開前チェック

- npm run check
- npm run build:public
- git diff --check
- dist/adminが存在しない
- Public JSONが6つの所定位置にある
- Public画面にConsoleエラーがない
- TRPG検索・お気に入り・共有URLが動作する
- Light / Darkで表示できる
- PC / 390px / 360pxで横スクロールがない
- OGP画像とメタ情報が配置されている

## 既知の運用上の制約

- localStorageは端末間で共有されない
- Adminは認証を備えたWeb管理画面ではないため公開しない
- Public反映はExport JSONの配置とcommitが必要
- Preview版のため、データ形式変更時はschemaVersionとdocsを同時更新する
