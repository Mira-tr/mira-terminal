# Public Theme Specification

Public theme is generated from theme data.

Public does not contain Studio theme editor state.

## Responsibilities

- Apply brand tokens.
- Apply creator theme layer when applicable.
- Preserve contrast.
- Preserve focus visibility.
- Support light and dark modes.
- Respect reduced motion.

## Validation

Before publishing, theme diagnostics should check:

- Text contrast.
- Button contrast.
- Focus-visible style.
- Responsive overflow risk.
- Motion reduction support.
