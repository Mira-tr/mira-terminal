# Creator Overview

Creators are independent public identities under RELMUA.

They are not Brand pages.

## Creator Owns

- Creator Home.
- Profile.
- Works.
- Contact.
- Creator-owned Collections.

## Creator Fields

Common fields:

- id.
- slug.
- displayName.
- shortIntro.
- bio.
- avatar.
- links.
- visibility.
- order.

## Validation

- id unique.
- slug unique.
- displayName required.
- links use http or https.
- private creators must not appear in Public index.
