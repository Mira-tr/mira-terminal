# Creator Modules

Creator modules are content systems owned by a creator.

TRPG under Chikage is the current example.

## Rules

- A creator module belongs to one creator unless a future contract says
  otherwise.
- Brand may link to creator modules but does not own their internals.
- Creator module routes are generated from creator context.
- Creator module data must not be duplicated into Brand data.

## Module Display

Creator pages may show:

- Module title.
- Short explanation.
- Primary action.
- Public status.

They should not dump every internal item when it makes the page too long.
