# Custom Collection Specification

Custom Collection supports future structured content.

It is allowed only when common types do not fit.

## Required Contract

A Custom Collection must define:

- Type ID.
- Owner.
- Schema.
- Validation.
- Preview.
- Public mapping.
- Backup behavior.
- Import/export behavior.
- Diagnostics.

## Field Rules

Custom fields must declare:

- Required or optional.
- Type.
- Length.
- Allowed values.
- Public mapping.
- Migration behavior.

## Safety Rule

Custom Collection must not bypass validation or public safety rules.
