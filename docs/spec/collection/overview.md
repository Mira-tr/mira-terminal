# Collection Overview

Collection is the Library system.

A Collection is a structured set of content.

TRPG is Collection Type 1.

## Collection Types

- TRPG.
- Game.
- Tool.
- Custom.
- Future types such as Book and Music.

## Rule

Collection architecture must not replace or break existing TRPG.

Existing TRPG JSON, Export, Public URL, search, Favorite, Filter, and House
Rules remain compatible.

## Lifecycle

Collections follow:

```text
Idea -> Draft -> Private -> Preview -> Published -> Deprecated -> Archived -> Removed
```

## Owner

Every Collection has an owner:

- Brand.
- Creator.

Chikage TRPG is creator-owned.
