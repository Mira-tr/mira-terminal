Operations Guide

Recommended order: edit, save, validate, export, backup, build, preview, publish preparation.

Production Flow

  1. Open the correct Brand, Creator, or System workspace.
  2. Save edits and resolve validation errors.
  3. Run Public Export for changed modules.
  4. Create a System Backup before risky operations.
  5. Run node scripts/build-public.mjs.
  6. Open Publish and confirm the Build Manifest and preflight results.
