House Rules / Chikage

卓をひらく前に、
一枚のしおりを。

ここにあるのは、物語を制限するためではなく、安心して物語の中へ入るための合図です。

Rules

基本方針

千景の個人卓では、以下の方針でTRPGを行います。

  • ルールブックの基本ルールを尊重する
  • 不明点はGMの裁定に従う
  • プレイヤーの提案を柔軟に検討する
  • 楽しくプレイすることを最優先する

System

使用ルール

主に以下のルールブックを使用します。

  • クトゥルフの呼び声 第6版（CoC6）
  • クトゥルフの呼び声 第7版（CoC7）
  • エモクロア コアルールブック
  • その他、シナリオに応じたルール

Character

キャラクター作成

能力値

基本的にルールブック通りのロールで決定します。ただし、GMの許可があればポイントバイ方式も可能です。

技能

技能ポイントはルールブック通りに配分してください。職業技能は職業に応じたものを選択します。

技能上限

初期段階での技能上限は以下の通りです。

  • 職業技能：最大75%
  • 興味技能：最大50%
  • ただし、シナリオによって上限が変更される場合があります

Sanity

狂気

SAN値チェック

ルールブック通りのSAN値チェックを行います。失敗した場合の狂気症状は、GMが状況に応じて決定します。

一時的狂気

一時的狂気になったPCは、GMの指示に従ってロールプレイを行います。他のプレイヤーも協力して演出に参加してください。

不定の狂気

不定の狂気になった場合、そのセッション中は回復しません。セッション終了後に治療が必要です。

Combat

戦闘

イニシアチブ

戦闘開始時のDEXロールで決定します。同値の場合はDEXの高い順、それも同じ場合はGMの裁定で決定します。

射撃

射撃武器の使用には技能ロールが必要です。失敗した場合、弾薬を消費しても命中しません。ファンブルの場合はジャムや誤射の可能性があります。

部位狙い

部位狙いを行う場合、通常の難易度に加えてペナルティを受けます。具体的な難易度はGMが状況に応じて決定します。

かばう

「かばう」アクションを行うには、DEXロールか技能ロールが必要です。成功した場合、対象が受けるダメージを肩代わりします。

致死ダメージ

致死ダメージを受けた場合、即死チェックを行います。失敗した場合、そのPCは死亡します。

Rating

年齢区分

全年齢（All）

一般的なTRPGシナリオです。年齢制限はありません。

R18

18歳未満は閲覧・参加できません。暴力描写や性的描写などの細かな注意要素はタグで案内します。

Notice

卓参加時の注意

時間厳守

開始時間は厳守してください。遅刻の場合、途中参加となる可能性があります。

環境

騒音の少ない環境で参加してください。マイクの使用は推奨します。

マナー

他のプレイヤーへの敬意を持ってロールプレイを行ってください。不快な言動は慎んでください。

休憩

適宜休憩を挟みます。休憩中は雑談や準備を行ってください。

キャンセル

やむを得ずキャンセルする場合は、事前にGMへ連絡してください。
